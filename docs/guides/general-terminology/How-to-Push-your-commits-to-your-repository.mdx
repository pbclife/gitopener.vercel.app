# How to Push your commits to your repository?

To push your commits to your repository in GitHub, follow these steps:

- Open your terminal or command prompt and navigate to the repository directory.
- Use the `git add` command to stage the changes you want to commit.
- Use the `git commit` command to commit the changes and add a commit message.
- Use the `git push` command to push the changes to your GitHub repository.
