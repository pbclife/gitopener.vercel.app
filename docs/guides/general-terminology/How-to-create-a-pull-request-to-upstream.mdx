# How to create a pull request to upstream?

To create a pull request to upstream in GitHub, follow these steps:

- Fork the upstream repository to create your own copy of the code.
- Create a new branch to make your changes in.
- Make your changes and commit them to your branch.
- Push your changes to your GitHub repository.
- Navigate to the upstream repository on GitHub and click the "New pull request" button.
- Select your forked repository and the branch you made changes in.
- Review the changes and submit the pull request for the upstream repository to review and merge.
