## Add your profile

There are 2 ways you can add your profile :

1. Directly from the GitHub
2. Using a code editor (eg. VSCode)

#### Step 1:

1. Visit the <Link href="https://github.com/pbclife/gitopener.vercel.app">GitOpener repo</Link>

<img
  src="https://github.com/pbclife/gitopener.vercel.app/blob/main/public/images/getting-started/repo.png"
  alt="GitOpener GitHub repo"
  style="border-radius: 10px"
/>

2. Fork the repo into your account

<img
  src="https://github.com/pbclife/gitopener.vercel.app/blob/main/public/images/getting-started/fork.png"
  alt="Fork GitOpener repo"
  style="border-radius: 10px"
/>

3. Click add file and select "create new file"

<img
  src="https://github.com/pbclife/gitopener.vercel.app/blob/main/public/images/getting-started/addFile.png"
  alt="GitOpener add File"
  style="border-radius: 10px"
/>

4. Create your file named `contribution/<Your GitHub username>`, and append `.mdx` (for example `contribution/Aadarsh805.mdx`)

<img
  src="https://github.com/pbclife/gitopener.vercel.app/blob/main/public/images/getting-started/createProfile.png"
  alt="GitOpener create Profile"
  style="border-radius: 10px"
/>

4. For the content use the mdx structure below:

---

author: Aadarsh Thakur
github_username: Aadarsh805
occupation: Student

---

# Introduction

Hi I'm Aadarsh Thakur, a frotend developer from Indore, India. I am currently a 3rd year Computer Science student persuing B-Tech.
I love building cool projects, contributing to OpenSource and getting involved in awesome tech communities.

## Skills

<Skills>
  <Skill
    name="Next Js"
    icon={/* svg */}
  >
    Lorem ipsum dolor sit amet consectetur, adipisicing elit. Sed, dicta!
  </Skill>
</Skills>

5. Add Projects

6. Click `Create pull request`

7. Now you need to wait to have your Pull Request reviewed
